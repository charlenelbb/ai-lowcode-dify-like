# 快速入门指南

## ⚡ 5分钟快速开始

### 1️⃣ 环境检查
```bash
node --version  # 应该 >= 16.0.0
npm --version   # 应该 >= 8.0.0
```

### 2️⃣ 启动项目
```bash
cd /Users/charlene/ai-lowcode-dify-like
npm install --legacy-peer-deps
npm run dev
```

### 3️⃣ 打开浏览器
访问 `http://localhost:5173`

## 🎨 编辑器基础操作

### 添加节点
1. 从左侧 **节点库** 拖拽节点类型
2. 放到中间 **画布** 区域
3. 节点即刻创建，可拖拽移动

### 编辑节点
1. **点击节点** 选中（显示蓝色边框）
2. 右侧 **属性面板** 自动显示配置选项
3. 修改标题、提示词等参数
4. 变化实时生效

### 删除节点
- 点击节点右上角的 **🗑️ 垃圾桶图标**
- 或选中后按 Delete 键（待实现）

### 缩放和平移
- **缩放**：鼠标滚轮向上/向下
- **平移**：点击空白区域拖拽画布

## 💡 常见任务

### 创建 AI 问答流程
```
输入 → AI 提示 → LLM 模型 → 输出
```

1. 拖拽 **输入** 节点到左侧
2. 拖拽 **AI 提示** 节点到中间
3. 编辑提示词："你是一个友好的助手，用中文回答问题"
4. 拖拽 **LLM 模型** 节点，选择 GPT-4
5. 拖拽 **输出** 节点到右侧
6. 点击工具栏 **保存** 按钮

### 创建条件分支
```
输入 → 条件判断 ├→ 路径1
              └→ 路径2 → 输出
```

1. 创建条件节点，输入表达式：`score > 80`
2. 连接上游数据源
3. 分别创建两条处理路径

### 数据清洗流程
```
输入 → 数据过滤 → 数据转换 → 输出
```

1. 使用 **数据过滤** 移除不需要的字段
2. 使用 **数据转换** 标准化格式
3. 输出处理后的数据

## 📁 文件操作

### 导出工作流
1. 点击工具栏 **📤 导出** 按钮
2. 浏览器自动下载 JSON 文件
3. 文件格式：`workflow_name.json`

### 导入工作流
1. 点击工具栏 **📥 导入** 按钮
2. 选择之前导出的 JSON 文件
3. 工作流自动加载（待完全实现）

### 本地保存
- 工作流自动保存到浏览器 LocalStorage
- 刷新页面后数据不丢失

## 🆘 故障排除

### 节点无法拖拽
**问题**：节点点击后无法移动
- 确保鼠标在节点内松开
- 尝试刷新页面

### 属性面板不显示
**问题**：选中节点后右侧没有显示配置面板
- 点击节点的中央区域（不是删除按钮）
- 检查节点边框是否为蓝色（表示已选中）

### 画布不响应
**问题**：无法缩放或平移
- 确保鼠标在画布区域（不在侧边栏）
- 尝试在空白处单击后再操作

### 工作流无法保存
**问题**：保存后刷新页面数据丢失
- 检查浏览器是否允许 LocalStorage
- 尝试导出为 JSON 文件备份
